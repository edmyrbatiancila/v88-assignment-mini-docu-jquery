My JQuery Mini Documentation

Click

.click() Binds an event handler to the "click" JavaScript event, or trigger that event on an element. Example: Clicking the button .click() will display alert box.

Hide

.hide() Hides the matched elements.Example: Hiding this paragraph by pressing .hide().

Show

.show() Displays the matched elements. Example: Showing the hidden paragraph here by pressing .show() button.

Toggle

.toggle() Display or hide the matched elements. Example: Show or hide this paragraph by pressing .toggle().

Slide Down

.slideDown() Display the matched elements with a sliding motion. Click the .slideDown button for an example

Slide Toggle

.slideToggle() Display or hide the matched elements with a sliding motion. Click button .slideToggle for an example to hide and show this sliding cat in sliding motion.

Fade In

.fadeIn() Display the matched elements by fading them to opaque. Click the .fadeIn() button to display an image.

Fade Out

.fadeOut() Hide the matched elements by fading them to transparent. Click .fadeOut() button to hide this cat.

Add Class

.addClass() Adds the specified class(es) to each element in the set of matched elements. Pressing the .addClass() button will add some element into it, please try.

Before

.before() Insert content, specified by the parameter, before each element in the set of matched elements. Noticed this sentence: "Viola!!" will appear before this sentence:"Magic!!!" when pressing .before(). Have a try.

After

.after() Insert content, specified by the parameter, after each element in the set of matched elements. By pressing .after(), noticed how new setence will keep adding after this message.

Append

.append() Insert content, specified by the parameter, to the end of each element in the set of matched elements. Set this as an example, when pressing the .append() a sign "!" will add in sentence: "I learn so much in Village 88"

Html

.html() Get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element. Pressing the button .html() will make the title have background-color and color elements.

Attr

.attr() Get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element. Click .attr() to change the photo below:

Val

.val() Get the current value of the first element in the set of matched elements or set the value of every matched element. If you press .val(), value of the input will change.

Text

.text() Get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements. Pressing the .text()change the text with p element.
